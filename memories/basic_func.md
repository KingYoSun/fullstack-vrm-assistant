# AI秘書の基本機能

- 音声対話: マイク入力を STT で文字起こしし、RAG で文脈検索→LLM 応答→TTS で音声出力までストリーム処理。
- アバター表示: three-vrm で VRM アバターを描画し、TTS 音量に応じてリップシンクを行う。
- 記憶検索: `memories/` などに保存した断片を FAISS 経由で検索し、関連文脈を LLM に渡して回答の正確性を高める。
- セッション管理: WebSocket セッションごとに会話ログやリクエスト ID を付与し、復旧やトレースに使える。
- 診断系: STT/LLM/TTS/Embedding/RAG/DB のヘルスを `diagnostics` エンドポイントで個別検証できる。
- フォールバック: プロバイダが応答しない場合は簡易フォールバックで動作継続し、ログに警告を残す。
